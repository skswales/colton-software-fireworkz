Fireworkz - Help

Install the Fireworkz-Manuals package to obtain the Getting Started Guide and User Guide.

Install the Fireworkz-Examples package to obtain the tutorial files referred to in the Getting Started Guide.

Online Resources

Many resources for Fireworkz are freely available online:

  • Getting Started Guide;
  • User Guide;
  • Addenda and Errata - updates since the documentation was printed, and errata therein;
  • Hints and Tips;
  • Downloads - updated packages for RISC OS and example files, including those used in the Getting Started Guide; also Fireworkz for Windows and Fireworkz source code ZIP archive sets. You are also welcome to modify the source code of Fireworkz as you want to meet your needs. The Mozilla Public License also gives you the right to distribute your modified versions.

Support

As Fireworkz is now completely free to use, support is a self-help affair. There is a Google Group colton-software-fireworkz where you may seek help from other users:
web site: https://groups.google.com/group/colton-software-fireworkz
e-mail: colton-software-fireworkz@googlegroups.com
Please include files which demonstrate any problems (using a Spark or ZIP archive).
