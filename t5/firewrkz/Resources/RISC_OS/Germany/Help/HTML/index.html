Fireworkz - Hilfe

Installieren Sie das Paket Fireworkz-Manuals um die Getting Started Guide und User Guide zu erhalten.

Installieren Sie das Paket Fireworkz-Examples um die Tutorial-Dateien zu erhalten, auf die im Getting Started Guide verwiesen wird.

Internetquellen

Viele Ressourcen für Fireworkz sind online verfügbar:

  • Getting Started Guide;
  • User Guide;
  • Addenda and Errata - Aktualisierungen seit dem Druck der Dokumentation und Errata darin;
  • Hints and Tips;
  • Downloads - aktualisierte Pakete für RISC OS und Beispieldateien, einschließlich der im Getting Started Guide verwendeten; auch Fireworkz für Windows, und Fireworkz-Quellcode-ZIP-Archivsätze.

Unterstützung

Da Fireworkz jetzt völlig kostenlos ist, ist Support eine Selbsthilfeangelegenheit. Es gibt eine Google-Gruppe colton-software-fireworkz, in der Sie möglicherweise Hilfe von anderen Nutzern anfordern können:
Webseite: http://groups.google.com/group/colton-software-fireworkz
e-mail: colton-software-fireworkz@googlegroups.com
Bitte fügen Sie Dateien bei, die Probleme aufzeigen (mithilfe eines Spark- oder ZIP-Archivs).
