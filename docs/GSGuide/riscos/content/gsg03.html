3 The basics

This chapter will describe some of the fundamental features of Fireworkz on RISC OS.

It includes sections on:

  • The mouse.
  • The document window.
  • The title bar.
  • The toolbar.
  • The status line.
  • The rulers.
  • Menus.

The mouse

The three buttons on your mouse each perform separate functions, and their names reflect this. Their names are:

  • Left mouse button: Select
  • Middle mouse button: Menu
  • Right mouse button: Adjust

Many actions using the mouse are performed by clicking or double-clicking an object. To do this, position the pointer over the required object (e.g. the Fireworkz application icon) and press and release Select (the left mouse button) once or twice respectively.

Pressing Menu (the middle mouse button) with the pointer inside a Fireworkz window displays the main application menu, from which you may choose an option by pointing at it and clicking with the Select button.

Adjust (the right mouse button) can be used to access some short-cuts and more sophisticated functions.

The document window

When you use Fireworkz to create a new document or load an existing document, the document will appear in a window on your desktop. For example:

The central area in white represents the paper onto which you can enter your document. Anything inside this area will appear when your document is printed, anything outside this area will not appear.

The other areas of the window are explained briefly below.

The title bar

The title bar of your document window includes some helpful information about your document:

  • If you have saved the document, its full file and path name is shown.
  • If the document is new, the name of the template on which it is based is shown. (Templates are used to defined how a document will be displayed - some templates are provided with Fireworkz and you can create others yourself. More information about use of templates is given in the User Guide.)
  • If the document has been modified since it was last saved, an asterisk will be displayed.
  • If you have a number of open windows belonging to the same document, the title of the document will contain a numeral indicating the number of views.
  • If the document is not being viewed at full size (i.e. at a scale other than 100%), the scale being used in that view will be displayed.

The title bar also contains buttons at the left and right to control the window:

  • Back button. Used to send the window to the bottom of the display stack.
  • Close button. Used to close the window. When closing the last view of a modified Fireworkz document, you will be asked whether or not you wish to save the document before closing it, or simply discard your changes and close the window.
  • Iconise button. Used to remove the window from the display to the Pinboard.
  • Toggle Size button. Used to make the window as large as possible on the display, and then to restore its original size from maximised.

Please refer to the RISC OS User Guide or the RISC OS online documentation.

The toolbar

The toolbar along the top allows you to access all the most commonly used commands quickly and easily by a single click on a tool button.

Greyed out (disabled) tool buttons

From time to time, certain tool buttons may be greyed out to indicate that they cannot be used at the moment. For example, you cannot use the Cut tool button to remove an area of your document uness you have an area selected. Therefore, if you do not have an area selected, the tool button will be greyed out.

As soon as you select an area, the tool button will become available for use.

'Pressed in' tool buttons

Certain tool buttons may appear to be 'pressed in' or 'pushed out' to indicate whether or not particular options are currently selected. For example, if the Bold tool button appears 'pressed in' (i.e. bold is currently selected)

then any text which you type will appear in a bold font.

If the tool button appears 'pushed out' (i.e. bold is not currently selected)

then text will appear in normal font weight.

If you click the Bold tool button, it will alternate between these two states.

The toolbar in Fireworkz Pro

The toolbar will change depending on the type of work you area doing. If you are in the middle of writing a letter, the word processing and spreadsheet tool buttons will be available. However, if you are currently within an area of your document which contains a database, then the database tool buttons will appear.

The status line

The status line appears below the toolbar. It provides help and information about the work in progress. For example:

  • If you point the mouse pointer over control areas of the window, such as the toolbar, a message appears explaining the function of the tool button.
  • If you drag the paragraph markers in the ruler. the precise measurements are shown in the selected unit of measurement, enabling you to align text and paragraphs precisely.
  • If you perform certain operations, such as word counts, the results are displayed on the status line until the mouse pointer is moved again.
  • Fireworkz Pro: If you click in a database cell within your document, the number of records in the database will be displayed.

Whilst you are familiarising yourself with how Fireworkz works, you should make use of the help provided by the status line.

The rulers

There are two rulers that you can choose to display:

  • A horizontal ruler along the top of the document.
  • A vertical ruler down the left-hand side of the document.

Horizontal ruler

The horizontal ruler can be used to alter the:

  • Paragraph margin - this determines the position of the left-hand edge of the first line of a paragraph.
  • Left margin - this determines the position of the left-hand edge of subsequent lines of a paragraph.
  • Right margin - this determines the position of the right-hand edge of a paragraph.
  • Tab positions - these determined where text will appear when it is preceded by one or more Tab characters.

Vertical ruler

The vertical ruler can be used to alter the:

  • Header margin - this determines the distance between the top of the paper and the printable area.
  • Header offset - this determines the distance between the top of the printable area and the header text (if present).
  • Footer margin - this determines the distance between the bottom of the paper and the printable area.
  • Footer offset - this determines the distance between the bottom of the printable area and the footer text (if present).

Menus

If you point within the document window and click the Menu button, the Fireworkz menu will appear.

Several menus contain sub-menus (or simple dialogue boxes); these are indicated by the right-pointing arrow alongside the option name. These sub-menus may be accessed by sliding the mouse pointer off the right-hand side of the menu past the right-pointing arrow; this will cause the sub-menu (or simple dialogue box) to be displayed.

Other options can be chosen by pointing at them so that they are highlighted and then clicking Select.

If an option has an ellipsis (…) to its right, this indicates that the command needs to pop-up a more complex dialogue box that cannot operate as a standard RISC OS Window Manager dialogue-box-as-menu. Typically these will be dialogue boxes which present lists of items for selection. The ellipsis on such menu options is omitted in this document for brevity.

Some frequently used menu options have keyboard shortcuts which are shown alongside the option name in each menu.

Greyed out (disabled) menu options

From time to time, certain menu options may be greyed out (disabled) to indicate that they cannot be used at the moment. For example, you cannot use the Sort option to sort an area of your document unless you have an area selected. If you do not have an area selected, the Sort option will be greyed out. As soon as you select an area, the Sort option will no longer be greyed out.
