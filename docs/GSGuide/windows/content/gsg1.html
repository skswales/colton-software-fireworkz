1Introduction

About Fireworkz

Fireworkz is part of the Z family of applications from Colton Software. This family includes the stand-alone applications Wordz, Resultz and Recordz. These stand-alone applications were developed for systems with very little memory in the early 1990s and are no longer available separately.

Wordz

Wordz could be used alone as a word processor to write letters, reports, books, invitations, shopping lists, schedules and other documents.

Resultz

Resultz could be used alone as a spreadsheet to perform calculations, analyse figures, and produce colourful charts.

Recordz

Recordz could be used alone as a database to store and manage all kinds of data from school and business information to personal details about projects and collections.

Fireworkz

Fireworkz combines the features of Wordz and Resultz enabling you to do even more; incorporate live results in your reports, or perform an automatic mail-merge to your friends or customers, for example.

Fireworkz Pro

Fireworkz Pro further combines the database features of Recordz with the word processing features of Wordz and the spreadsheet facilities of Resultz enabling you to create fully integrated documents. Fireworkz Pro is only available on RISC OS.

Once you have learned to use one of these programs you will find it easy to use the other members of the family, because they share much of the same interface and have been designed to work well together. This is true even if you move from RISC OS to Windows or vice versa; the programs have been designed to be as similar as possible on each platform.

About this Getting Started Guide

This Getting Started Guide includes all the information you need to start using Fireworkz on Windows.

  • A brief installation guide;
  • Tutorials using supplied documents to show you the main features of the program;
  • Hints and tips for transferring files between the RISC OS and Windows versions of Fireworkz.

You can find more information about the program and its features in the on-line help and in the User Guide.

Knowledge assumed

This Getting Started Guide assumes that you are already reasonably familiar with the way your computer works. If you are new to computing, you are advised to familiarise yourself both with the relevant terminology and the basics of using your computer. You will need to know how to:

  • Start up programs and open documents;
  • View directories and copy files;
  • Display menus and choose entries from them;
  • Position and resize windows, minimise and restore windows;
  • Scroll through documents using the scroll bars and arrows;
  • Use the mouse to move the pointer around the screen and to position the caret;
  • Use the mouse buttons to click, double-click and drag objects.

You should understand terms such as directory, double-click and dialogue box.

You should be able to find all this information in the system documentation which came with your computer. The tutorial chapters in this Getting Started Guide also provide a reminder of these skills.

Text in this Getting Started Guide

When working your way through the tutorial chapters, you will see that certain stylistic conventions are used to make it easier for you to see what you should do at each stage, what you should type in and what you should choose from menus.

Certain buttons, options and commands always appear in the text in bold type. These include:

  • Menu options. For example:
    ‘Choose the Print command from the File menu.’
  • Action and option buttons in dialogue boxes. For example:
    ‘Click the OK button.’
  • Buttons on the toolbar. For example:
    ‘Click the Style button.’

In some places there is text for you to type in. This is always shown in a typewriter-style font, like this:

Type this text on your computer exactly as you see it.

When the text given represents what you will type, for example, the name to be given to a file before saving it, this is shown by using italics, such as:

Type filename in the dialogue box which appears.

There are several keyboard shortcuts which you can use in Fireworkz.

If you see a key combination such as Ctrl-X or Alt-space, it means you should press the keys together in the following way:

  1. Press and hold down the first key of the pair.
  2. Press and release the second key.
  3. Release the first key.

Some commands can be accessed directly by pressing a function key . Where this is the case the function key is referred to by the name shown on the key cap itself; for example function key F1.

You will see many references to the caret. This is the special pointer which shows you where you are in a document, and where any new text you add will be placed.

Conventions used

There are set terms used to describe many of the parts of your computer, the objects displayed on screen and tasks you will carry out.

The mouse

Fireworkz on Windows uses the left button of the mouse (or its equivalent on any alternative pointing device which you use). When you are called to click on an object, this means click and release the left button of the mouse. For some special operations (those which are performed on RISC OS with the right button of the mouse) you will need to press and hold the Ctrl key whilst clicking the left button of the mouse on Windows.

The window

The following terms are used for the icons and buttons with which you control windows:

  • Window title bar.
    This includes some helpful infomation about your Fireworkz document:
    • If you have saved the document, its file name is given;
    • if the document is new, the name of the template on which it is based is given (more information about use of templates is given in the User Guide);
    • If the document has been modified since it was last saved, an asterisk will be added;
    • if you have a number of open windows belonging to the same document, the title of the document will contain a numeral indicating the number of views;
    • if the document is being viewed at a scale other than 100%, the scale being used in that view will be added.
  • Minimise button.
    Used to remove the window from the display to the Windows taskbar.
  • Maximise button.
    Used to make the window as large as possible on the display.
  • Close button.
    Used to close the window.
    When closing the last view of a modified Fireworkz document, you will be asked whether or not you wish to save the document before closing it, or simply discard your changes and close the window.
