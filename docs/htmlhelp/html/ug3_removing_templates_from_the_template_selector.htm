Removing templates from the template selector

RISC OS: Use the Filer to access the directory !Fireworkz.Resource.Template or Choices:Fireworkz.Template and delete or rename the template files you no longer want.

Windows: Use the Windows Explorer to access the directory %USERPROFILE%\Application Data\Colton Software\Fireworkz\Template and delete or rename the template files you no longer want.
