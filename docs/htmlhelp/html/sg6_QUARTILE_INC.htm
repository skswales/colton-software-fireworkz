QUARTILE.INC

Returns a quartile value from a data set.

Syntax:

QUARTILE.INC(array, q:number)

Returns the qth quartile from the values in the array or range of cells based on a percentile range of [0.0,1.0] (i.e. inclusive).

q must be of one the following values:

0  0% = MIN

1  25% = Q1

2  50% = Q2 (MEDIAN)

3  75% = Q3

4  100% = MAX

Added in 2.00.
