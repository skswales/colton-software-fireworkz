INDEX

1. INDEX(array, col:number, row:number)

Returns the element of the array or range of cells specified by col,row. Column and row indices start at 1.

Example:

INDEX(B1:D20, 2, 3)

returns C3.


2. INDEX(array, col:number, row:number, cols:number, rows:number)

INDEX with five parameters returns a sub-array with cols columns and rows rows starting at the position col,row.

Example:

INDEX(B1:D20, 2, 3, 2, 5)

returns C3:D7 as an array.
