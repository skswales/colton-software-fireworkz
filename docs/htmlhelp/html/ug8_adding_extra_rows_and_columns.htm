Adding extra rows and columns

There are two options for adding extra rows and columns to a table. You may either add rows and columns to the end of an existing table, or insert them at a point within the table.

To add rows or columns to the end of a table:

  1. Place the caret in the last row or column of the table
  2. Display the Columns or Rows sub-menu as required from the Edit menu.
  3. Choose Add from the Columns or Row sub-menu. The Add Columns or Add Rows dialogue box will be displayed.
  4. Enter the number of columns to add by typing in the box or using the arrows.
  5. Click OK. The extra rows or columns will be added. They will be displayed in the table style for this table.

To insert extra columns or rows within a table:

  1. Place the caret within the table at the point where you want to insert an extra row or column.
  2. Display the Columns or Rows sub-menu as required from the Edit menu.
  3. Choose Insert from that sub-menu.
  4. An extra column or row will be inserted into the table.
