Setting up paper size

One of the most important options which you need to defined for every worksheet is the page size and printable area. This will ensure that you can print it correctly.

The end result of most documents which you create will be a printed document. In order to ensure that all your documents print out correctly first time on your printer, without text being lost off the edge of pages, you should read the following sections.

If the page and margin sizes that Fireworkz uses on-screen do not match those of your printer driver, you may have problems when you try to print your document.

The simplest way to ensure that your document and printer driver match is to use the Read printer option in the Paper dialogue box. This matches the printable area shown on screen with the printable area produced by your printer. In effect, it makes it impossible for you to format your pages incorrectly. If you know your document is always going to be printed on the same printer, you should use this for maximum accuracy.

However, the Read printer option will only be accurate if your printer driver has been set up carefully. Remember to refer to your operating system and specific printer documentation.

The key thing to remember is that if you attempt to print a document at full size on a piece of paper smaller than that it has been designed for, or using a printer which can only print a smaller area, you will not get the full page on the piece of paper.

If you find that you are getting page feed or line feed problems (such as extra blank pages), it is most likely that your printer driver has the wrong page length set. Try reducing the page length in your printer driver, if that is possible, then click on Read printer again in Fireworkz.
