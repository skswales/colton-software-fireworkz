DMAX

Syntax:

DMAX(range, condition)

Returns the maximum value in the range for which the condition is satisfied.

Example:

DMAX(A1:C5, A1:C5>5 & A1:C5<70)

will return the largest value from the range of cells A1 to C5 which is greater than 5 and smaller than 70.

See DAVG for more examples of how to use the condition to control which cells are included.
