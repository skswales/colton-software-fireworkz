Colour

If you will be using a colour printer, or will be using your document mainly on screen, you can set the colour option and use a colour other than black for your text.

  1. Click the colour switch to on. The colour section of the dialogue box will become illuminated.
  2. Choose the colour you want. There are three ways to do this: you can click on one of the small buttons representing the colours in the current palette, type in numerical RGB values, or use the arrows to enter RGB values. On Windows, click on the Colour button to access the colour selector to choose a colour or define a custom colour. Note that if you are using a 16-colour screen mode you will see on the screen only those colours in the current palette, whatever numbers you specify as RGB values.
  3. The colour you have chosen is shown in the bar at the bottom of the colour section.
