Upper-case and lower-case

There are several ways to change the case of selected text. You may change text to upper-case, lower-case, or initial capitals, or simply swap case.

  • Use the appropriate Case option from the Edit menu.
  • There is a key short-cut for swapping case. Select the words whose case you want to change and press Ctrl-S (RISC OS).
  • If you only want to change a single letter, place the caret before the letter and press Ctrl-S (RISC OS).
