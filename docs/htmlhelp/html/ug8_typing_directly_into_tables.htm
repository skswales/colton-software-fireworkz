Typing directly into tables

To type directly into a table, place the caret in the cell you want to type in and enter your text. This will enter the information as text.

Text typed into cells will automatically flow on to a new line if necessary, creating a deeper cell, unless Fixed row height is turned on. In this case the row height will not automatically expand to compensate for this, and you will need to accommodate the extra text by making the column wider or the row deeper using the row and column borders.

Press Ctrl-Return to start a new line within a cell. If you have several words to enter within a cell (perhaps a list of items, or parts of an address) this lets you ensure that each starts on a new line.

Use the arrow keys on the keyboard to move between cells. Or press Tab, Shift-Tab and Ctrl-Tab to move around the table.
