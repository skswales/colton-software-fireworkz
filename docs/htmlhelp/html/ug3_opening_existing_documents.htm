Opening existing documents

Much of the time you will want to re-open existing documents for further editing.

RISC OS: Simply double-click on a Fireworkz file's icon in a filer window to load it into Fireworkz. This will cause Fireworkz to be loaded, if it was not already.

Windows: Simply double-click on a Fireworkz file's icon (these will have the .fwk extension) directly in Windows Explorer to load it into Fireworkz. This will cause Fireworkz to be loaded, if it was not already.

Alternatively, if you already have Fireworkz loaded, choose Open from the File menu, or click on the Open document button. The Open dialogue box will appear:

Find the drive, directory and file you want. Choose the file format you want. Click OK.

The file will be opened in Fireworkz.
