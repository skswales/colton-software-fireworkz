X-Y Scatter plots

Scatter plots display data as discrete points and with or without lines between them.

To create a scatter plot, select your data in the worksheet and click the Chart button. Choose X-Y from the Chart menu and a dialogue box will appear.

  1. Click on the button for the type of scatter plot you want.
  2. Specify whether the data series are in columns or rows.
  3. Specify how the data series are arranged. X, Y, Y means that you want to plot more than one y-series against a single x-series. X, Y, X, Y means that for each y-series you are supplying an x-series.
  4. Specify whether the first series is data or labels.
  5. Click OK and your scatter plot will be drawn.

For details of scatter plots available, see Types of scatter plot.
