Creating templates

To save a file as a template, first set it up as described in the previous section then do the following:

  1. Choose Save template from the File menu. The Save template dialogue box will appear.
  2. Type in the name of the new template (ending in the extension .fwt if on Windows).
  3. Choose one of the options for what to include in the template by selecting the appropriate radio button. You can save the entire document (styles, other definitions, pictures and text), just the styles defined in the document, or a single style. If you choose to save a single style you should select one from the list by clicking on it. If you want to create documents based on the template, you must choose the All option.
  4. Click OK. The template will be saved in the directory <Choices$Write>.Fireworkz.Template (RISC OS) or %USERPROFILE%\Application Data\Colton Software\Fireworkz\Template (Windows). This means that the template appears on the template selector list when you open a new document.
  5. If you do not want to save the template in the standard location, you should type in the full pathname. On RISC OS, you can drag the file icon to the directory where you want it to be saved.
