Number formats and styles

This section describes how to work with the Number section of the Style dialogue box to define number, date and time formats.

It also contains a general introduction to using styles in spreadsheet documents, showing how you can use styles to apply formatting to text and numbers within the document, and to customise spreadsheet features such as the style used to show the current cell.

This section of the Style editor also contains other functions, such as marking cells as protected, so that they are locked against future editing.
