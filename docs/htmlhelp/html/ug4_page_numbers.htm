Page numbers

Page numbers work best if they are added to running text, that is headers and footers. If you do this, the correct page number will be displayed on each page of your document.

To add page numbers to a document:

  1. Put the caret in the place where you want the page number.
  2. Display the Inserts sub-menu from the Extra menu. Choose Page #....
  3. The Page number dialogue box will appear. Select the page number format you want and click OK.
  4. The correct page number will be displayed automatically. If you added the page number to a header or footer, the correct page number will appear on every page.

Restarting page numbers

To restart page numbering:

  1. Enter a page break.
  2. While in the Page dialogue box click the Page number button.
  3. Type the number you want to start from in the box and click OK.

x-page numbers

In documents which have across pages as well as down pages (x-pages), such as wide spreadsheets whose columns run across more than one page, you may want to number these pages. To number across pages, choose X page #... from the Inserts menu.
