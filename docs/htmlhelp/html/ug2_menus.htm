Menus

The menus in Fireworkz appear at the top of the document window enabling you to choose commands from them as necessary.

Some menu entries include an arrow leading to a submenu from which you should make your choice. Others include a keyboard short cut which enables you to bypass the menu entirely. This is shown by a short cut code next to the menu entry.

You can also select entries from menus by typing in the underlined letter, preceded by Alt and the underlined letter from the menu title to select the menu
