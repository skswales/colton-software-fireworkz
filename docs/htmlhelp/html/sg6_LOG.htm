LOG

Syntax:

LOG(x:number {, b:number})

Returns gives the base ten logarithm of x.

x must be > 0.

Example:

LOG(64)

returns the number 1.80618.

Starting with 2.00, this function will accept an extra numeric parameter, b (optional), which then returns the logarithm of x to base b.
If b is omitted, the logarithm of x to base ten is returned as before.
