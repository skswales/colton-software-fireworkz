Printing filename within documents

You may find it useful to print the filename or pathname on your document, to remind you where it is stored.

Two options are available, both from the Inserts menu:

  • The pathname shows the full pathname of the file, showing the route to it from the filing system through any directories to the file itself, such as:

RISC OS: ADFS::HardDisc.$.SKS.Documents.July.Letter

Windows: C:\Users\SKS\Documents\July\Letter.fwk

  • The filename shows just the last part of the file's name, the name you typed in when you saved it. So the file above would appear simply as:

Letter

To use these options:

  1. Place the caret where you want to insert the name into your document.
  2. Display the Inserts menu from the Extra menu.
  3. Choose Filename or Pathname.
  4. The filename or pathname will be added to your document.
