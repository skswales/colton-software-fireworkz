SKEW

Returns the skewness of the values in a data set.

This is a measure of the asymmetry of a distribution of values about their mean.

Syntax:

SKEW(list)

Returns the skewness of the values in the list, which may be an array, a range of cells, or a comma-separated list of values.

Example:

SKEW(A1:A15)

returns the skewness of the contents of cells in the range A1 to A15.
