Getting started with spreadsheets

This chapter provides an introduction to Fireworkz and to basic spreadsheet features which it contains. If you are new to using Fireworkz or to using spreadsheets, you should work through this chapter.

Open the Tutorial directory using File Manager; you will see a file called Accounts. Load this file by double-clicking on its icon. You’ll see the following window:

The file contains the simplified accounts of a small business. Some categories of information have already been entered. However, not all the details have been filled in. We will be entering more text and numbers to the worksheet, and performing some calculations to add up the columns of figures.

Cells

Each piece of information appears in a box called a cell. Some cells contain text, and others contain numbers. The two types of cell behave in slightly different ways, as you will see when you work with them.

In this worksheet, the cells around the top and left edges of the page are text cells, and the others are number cells.

Cell references

Each cell has a unique address within the spreadsheet, formed by its column and row name. Within a worksheet each column and row have a unique heading letter or number, and these are used to identify each cell. For example, the cell at the intersection of column A and row 9 is referred to as cell A9. However you type them in, cell references appear in lower case in the formula line.

Many operations you carry out in a spreadsheet are done by entering a function in one cell, and giving it the cell references of the cell or cells containing the data. The function is a command which performs an operation on the data, such as adding it up, or working out the average. The cell containing the function then displays the result of performing that calculation on that data.
Entering text

When you enter text into a text cell, you type it straight on to the worksheet page.

To enter text in a cell you need to place the caret there, to make it the current cell.

Click once on cell A9. It is now the current cell. There are three ways you can tell this:

The row and column headings A and 9 are emphasised.

A border appears around cell A9.

You can see the caret in cell A9 (this only applies to text cells).

Type in the word Travel into cell A9 and press Return.

Now A10 is the current cell. Type in the word Total and press Return.
Entering numbers

When you enter numbers into number cells, you type them in the formula line and then click the Tick button or press Return. Then the formatted version of the number you enter is displayed in the worksheet cell.

As you can see, there are still a lot of blanks in the number section of the worksheet to fill in before we can calculate the totals.

Entering single numbers

Click on cell B9. You’ll see that it develops a border now that it is the current cell, but no caret appears. This is characteristic of number cells. Click in the formula line.

Type 60. The numerals don’t appear in the worksheet cell, but on the formula line at the top of the screen. When you enter data or formulas into number cells, they are always displayed on the formula line first, and entered when you press Return or click on the Tick button. When you are working on the formula line, many buttons which only apply when you are working on the main worksheet are ‘greyed out’:

Press Return. You’ll see 60.00 entered in cell B9 on the worksheet. Fireworkz has applied a format to the number, displaying it with two decimal places:

Enter some more numbers into the following cells:

Copying numbers

Cell E8 is still blank. However, the number to enter there, 40, is the same as the number in cell C8. Rather than type in a new number, we will copy the number from cell C8 to E8.

Select the contents of cell C8 by double-clicking in the cell. The whole cell will turn black to indicate that it is selected. Click on the Copy button on the button bar.

Click on cell E8. It now becomes the current cell.

Click the Paste button on the button bar. The number 40.00 will now appear in both cells.

Entering blocks of numbers

You will notice that we still have not entered the figures for the Rates in row 5. This is a fixed cost, the same every quarter. We don’t have to enter the number in every cell individually. Instead we can fill the cells.

Selecting a range

To fill a group of cells we must first select the group of cells to fill. Groups of adjoining cells are called ranges. Click in cell B5 and drag until cell E5 is also selected:

All the selected cells will appear solid black.

Filling cells

You will see that the two buttons at the top left of the Fireworkz window are no longer greyed out. These buttons enable you to enter the same number into several cells at once.

The data must already appear in the first cell in the range, as it does in cell B5.

Click the Fill right button on the button bar. You will see the number 150.00 appear in all the selected cells.

Adding numbers

There are some blank spaces left in the worksheet for the totals. Adding up columns and rows of numbers is one of the most common operations performed with spreadsheets. There’s a button on the button bar to enable you to total columns or rows automatically.

To do this, all you need do is select the cells you want to total, plus a blank cell at the end to display the result in.

We’ll add column B first.

Select cells B4 to B10 by clicking in B4 and dragging downwards.

Now click on the Autosum button at the top left of the button bar.

The figure 885.00 will appear in cell B10.

Do the same for columns C, D and E. You can speed this up even further, by selecting the range C4 to E10 and clicking Autosum. Fireworkz will automatically total each column.

Finally, we need to add the contents of rows 4 to 10. You can do this in one operation by selecting cells B4 to F10 and clicking Autosum. Column F will now contain the totals of each row, and in cell F10 the grand total.

If you click on cell F10, you will be able to see in the formula line the way Autosum works. The formula line reads:

sum(b10e10)

The function which performed the operation is mentioned first, followed by the cells on which it operated, in this case the range B10 to E10.

A slower but simpler method of getting the same result would have been to type in the formula line

b10+c10+d10+e10

But this would have taken much longer. Also, if you later insert extra columns into the range covered by the sum function, it automatically updates to include the extra columns.
Applying styles and formats

The worksheet is now full of data, but it is unformatted. We need to know more about numbers than the numerals to understand them; if they are money, what currency, if they are a percentage, and so on. Formatting the numbers makes them easier to understand because we know more about what the numbers represent, and about their size.

The numbers in this worksheet are, in fact, currency and should be formatted as such.

Formats are applied to cells using the Style system. This is a feature of Fireworkz which allows you to collect formatting information and give it a name, and then choose the name from the style list to apply to any section of your worksheet.

To apply a style to an area, you must first select the area.

Select the cells containing numbers, cells B4 to F10.

Click on the Style button. You’ll see a list of available styles. Several styles have already been defined in this document; it’s possible to create new styles if you need them, but the ones we need have already been set up.

Choose the style pounds pence from the list by clicking on it.

Click Apply. The numbers will appear reformatted with pound signs:
