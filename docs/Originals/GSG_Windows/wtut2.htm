What’s in this chapter

This chapter is intended to introduce you to some of Fireworkz’ powerful and advanced features. You should find that they are no harder to use than the basic features you learned in the last chapter.

In this chapter you will learn how to:

Add new styles and update existing ones using the Style dialogue box

Layer styles to provide a combination of changes to areas of text, including colour, emphasis and typeface.

Use Effects to provide one-off changes

Use the ruler and markers to update styles

Add a box to an area of text

Create and edit tables

Resize columns in tables

Add colour to tables

The example documents in this chapter were designed to work on A4 paper. If the printer you will be using does not print on A4 but a different length or width of paper, you will need to tell Fireworkz this before you start.

Ensure that your printer driver has been installed correctly.

Choose Paper... from the Page menu.

Click the button marked Read printer, then click OK.

This process will ensure that the document is formatted correctly for your printer, using the information stored in your printer driver. See the chapter in the User Guide on Printing for further details of setting up your printer driver.
Using the style editor

The style editor is one of Fireworkz’ most powerful features. It’s well worth learning to use to get the best out of the program. Styles are a powerful tool for formatting your text and documents. They enable you to tag similar things such as all the headings in your document, and apply the same typeface, size, and margins and so on to them.

This makes designing your document quicker, because you only have to set up a style once and then it can be quickly applied to any text, and ensures that your documents are consistent, because when you change a style Fireworkz automatically changes all text which appears in that style. This Getting Started Guide, for example, has styles set up for headers and footers, for headings and for body text and indented and numbered paragraphs.

Before you start to edit styles and create new styles, you should familiarise yourself with the style editor by looking at some of the styles in the document you’ll be working on.

Load the example document newslet.fwk. It is in the Tutorials directory in the firewrkz directory.

Click on the Style button. A dialogue box containing the list of styles is displayed.

Click on BaseLetter and then click Change.

This is the base style for the document.

The style editor is divided into seven sections which you can reach by clicking on the radio button next to their title. You’ll see that all the green buttons at the left edge of the style editor are illuminated; this indicates that something has been selected in that section of the style editor. They just help you find out what attributes a style has. The first green button is always on as every style has a name.

Styles only change those aspects of text which have actively been set in them. A style which makes text bold will not change the colour of text, or its size.

Click Cancel.

Follow the same procedure, but this time choose Bold from the style list.

You’ll see that this style has only one active section, Text. Applying this style to any text will change the type to bold, because the check box and the buttons are switched on. The check box indicates whether the style affects the boldness of text—in this case it does—and the button shows you what that effect is—it makes the text bold. If the right button were in the off position, the effect of the style would be to remove any occurrence of bold text it encountered.

Everything else, from margins to text size to colour, will be left the same.

You should spend some time looking at the options in the Changing style dialogue box. If you don’t understand any of the terms used, they are all defined in the Glossary at the back of the User Guide.

Adding a new style

The newsletter does not have a style for headings. Headings currently appear in the same style as the main text (referred to as body text).

To add a new style:

Click on the Style button, then click New. The following dialogue box will appear:

Type Heading in the space for the style name.

Click on the style BaseLetter in the list. It’s often easier to base new styles on an existing one.

Click OK. A new dialogue box will appear with the title Defining new style: Heading.

Click on the Text button, to go to the part of the style dialogue box where you can alter text style:

You will be making the type bold and increasing its size. So you need to click on the check box next to the bold button. Both it and the Bold button will be pressed in. Next, click on the height check box. Use the arrows to increase the size to 16 points.

Click OK in the bottom right corner of the dialogue box.

The new style is now added to the list of styles. If you click on the Style button you will see that Heading is now included in the list of styles.

Applying your new style to existing text

There are several headings already in the document to which you need to apply the Heading style. To do this:

Go to the first heading, and select it: ‘Latest news from your youth club and community centre.’

Click on the Style button.

Choose Heading from the list of styles and click Apply.

The heading will appear in bold and in larger text.

Go through the other headings in the document and do the same to them.

Choosing the style for text entry

One way to design your document is to enter all the text and then to add styles to it. However you may also choose the style for text before you enter it. You need to add another text paragraph to the newsletter:

Place the caret at the end of the paragraph ‘Thanks to everyone...’ and press Return. The caret is now at the start of a new paragraph.

Click on the Style button in the button bar.

Click on Bold and then Apply.

Type in:

Also thanks

Click on the style button again and choose BaseLetter, then click Apply. Type in:

to the Parent and Toddler group for raising £145 with the sponsored tricycle ride.

You will see that the first two words appeared in bold and the rest in the base text style.

Changing styles without using the dialogue box

There are some aspects of style which are much more easily set up or changed by working on page with the buttons and ruler markers than by going through the style dialogue box and typing in numbers. Setting margins and tabs are the main example.

The list of points at the bottom of the page need to be indented (moved towards the centre of the page) further than they are already. To change the style:

Place the caret in the line beginning ‘Junior disco’. But make sure that you don’t select any text; you can check this by looking at the Selection button in the button bar, which should be white.

Point at the left margin marker, and drag it one centimetre to the right. The paragraph margin marker and tabs will follow it:

All three paragraphs will reformat in the changed style.

If you attempt to change a style while there is a selection, your changes will be applied as an effect to the selected area. An effect is like a one-off occurrence of a style, but no record of it is stored in the Style list so it is harder for you to keep control of your document and go back and make stylistic changes.

Layering styles

One of the most powerful things you can do with styles is to combine them with each other to add further dimensions of style to your documents.

To make this easy, you should define as little as possible in each style. For example, styles changing the colour of text should not also change the typeface or text size. This makes it easy to build combinations of styles, giving you added flexibility.

You have already applied one style, Heading, to the headings in the document. Now go back to them and layer another style over them to add colour.

Select a heading and click on the Style button.

Choose the style Text colour from the list and click Apply.

The text will change to the new colour but remain in the same typeface.
Adding tables

Tables are a useful way of setting out complicated information which would be hard to understand if it was written in sentences. But Fireworkz’ table facilities offer much more than that. They allow you to create a wide range of custom forms and similar documents.

To work with tables, open the document Form in the Examples directory on the Examples disc.

Adding the table

Place the caret at the end of the text in the document and press Return to start a new paragraph.

Choose Table from the Edit menu. The Insert table dialogue box will appear:

Enter five columns and six rows, then click OK.

The table will be added to the document. At this stage it will be very plain and not include much of the style detailing which you are going to add.

Resizing columns

To get your table looking just right you will need to change the width of some columns.

The last column, ‘Preferred date’, needs to be quite wide because people will be writing in it. To widen the column:

Place the caret somewhere within the column.

Point at the column width marker in the ruler. The pointer will turn to the two-way arrow, and the status line will confirm that you are pointing at a column width marker and give you the measurement.

Press down the left mouse button and drag the marker to the right for about 2cm.

The column will be resized.

In some cases you will want to make sure that the column is just the right size for the text it contains. This feature is called Auto width. To resize the first column automatically:

Place the caret in the first column.

Point at the column width marker in the ruler. The pointer will change into a two way arrow.

Double-click on the column width marker.

The column will resize automatically to the correct width for the widest text it contains.

Adding colour and a special border

Tables can sometimes be difficult to read. Adding colour is one way of making it easier for people to keep their eyes on the right row or column.

To add colour to the table:

Create a new style, and give it a name like Table colour.

Go to the Colour section of the Define new style dialogue box, and click on background.

Choose a background colour for the table. Bear in mind that it will have text printed over it, so it should be a fairly pale colour.

Click OK.

Now select a column from the table, and click the Style button.

Choose Table colour from the style list and click Apply.

One possibility is to add colour to alternate columns or rows, or to groups of rows or columns.

Adding a box

You can add further definition to your table by adding a heavier rule around its outside. To do this:

Select all the cells in the table.

Choose Box from the Style menu:

Click the button Outside.

Click the thick line button in the line style section, then click OK.

A thicker line will be drawn around the outside of the table.

You can also use the Box dialogue box to add thinner lines within tables, or to change the colour of lines in tables.To remove lines from a table, choose the blank button in the Line type section of the Box dialogue box.
Saving your work

The final step in the tutorial is to save your work.

You should save your document on to another disc or in another directory.

Click the Save button.

The standard Save dialogue box will appear:

Enter a name for the document, such as-form.fwk. You must make sure that the file has the .fwk suffix as this enables Fireworkz to recognise it.

Choose the drive and directory where you want to save the file.

Click OK. The file will be saved.
