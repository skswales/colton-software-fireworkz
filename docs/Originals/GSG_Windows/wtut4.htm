Writing formulas

This chapter introduces formulas and shows you how to write them. It introduces the operators and functions which you need to use in formulas and shows you the ways Fireworkz can help you to write formulas easily.
Using functions

Load the worksheet exams.fwk from the Tutorial directory.. You will see a list of class members and their marks in four exams:

We need to calculate the average mark for each pupil, and to award them each a single grade based on this average mark. We then need to work out the number of pupils receiving each grade, and the percentage of the class receiving each grade.

The function selector

In the previous chapter you added columns of numbers using the Autosum button. You saw that this created a formula which you could read on the formula line. In this chapter, you will be entering formulas directly on the formula line.

Working out an average

The first step is to work out the average number of marks earned by each student.

Click in cell g3. We need to add the avg() function to this cell. You can type in functions directly, but we will use the function selector, an easy way of entering functions.

Click in the formula line; you’ll see the caret appear there.

Now click on the function selector button.

Choose Quick... from the function selector menu by clicking on it. You’ll see a short list of functions, including avg(list).

Double-click on the avg(list) entry. It will now appear on the formula line:

The word list is called the parameter to the function. You need to replace it with the actual data on which the function will operate. The avg() function needs a list of numbers to operate on. You could supply a list of individual cell references, but in this case we will supply a range, indicating that the function should operate on the data in all the cells contained between, and including, the two cell references given.

You could enter the range details by typing in the top left and bottom right cells in the range, but it’s easier to use the mouse.

Click on cell b3 and drag to cell e3. You’ll see these cell references appear between the brackets on the formula line; note that list has been replaced:

Your formula is now complete. Click on the Tick button or press Return to enter the formula.

You will see the result appear in the cell.

We need to perform the same calculation for all the students in the class. This can be done using one of the Fill buttons.

When you use a Fill button to copy a formula, it updates the formula in each row or column, so that the data it operates on has the same relative position to each cell containing the formula. So when our formula appears in row 4, it will calculate the average value in row 4, and so on.

Click on cell g3 and drag to select the cells in column g up to and including g18.

Now click on the Fill down button (because we are filling down a column). The column will fill with the average mark for each student:

The if function

The next stage is to assign a grade to each student, based on their average mark. The school operates the following formula:

                          Average mark                                      Grade

                          over 75                                            A

                          50-75                                               B

                          under 50                                          C

We could simply look at each mark and type in the appropriate grade. However, we can write a formula which calculates the grade, by using the if() function.

The aim of our formula is to look at a mark, and see if it is over 75. If it is, the letter A will be printed in the column. If it isn’t, the formula will look and see if the mark is between 50 and 75. If it is the letter B will be printed; if it isn’t (ie it is less than 50) the letter C will be printed.

To start writing the formula containing the if() function, click on cell h3 and then click on the formula line. Click on the Function selector button, then on Quick... on the menu which appears. Double-click on if(condition,then,else) and it will be added on the formula line.

Writing the condition

The if function has three parameters.

The first is the condition, the test which the formula applies to the mark. If the condition is true, the action detailed in the second parameter takes place. If it is false the action detailed in the third parameter takes place.

We need to test that the mark is greater than 75. To do this we simply enter a comparison as the first parameter. Fireworkz will take the contents of the cell, and test to see if the answer is true, or false (less than 75) and act accordingly:

g3>75

To enter the cell reference g3, simply point and click at that cell, then type in the rest of the condition >75. Our formula now reads:

if(g3>75,then,else)

Printing a string

Next we need to detail what will happen if the condition is true. We want the letter A to appear in the cell, so we simply need to type this in. Text manipulated by formulas needs to be surrounded by double quotes, so that Fireworkz can recognise it and process it correctly. It is referred to as a string, because it is a string of characters.

Enter "A" after the comma. As you move on to subsequent parameters, they will be highlighted; appropriate help will appear in the status line. As before, just type over the parameters to replace them. All parameters must be separated by commas, so that Fireworkz can tell them apart. The formula will now read:

if(g3>75,"A",else)

Nesting if functions

The last stage is to enter what happens if the average mark is less than 75. In this case, what will happen is that a further test will be applied to the mark, to see if it is between 50 and 75. To write this part of the formula, we need to use a further if() function. This process is known as nesting functions.

This time we will type in the function directly. Between the last comma and the close bracket, type over the else part:

if(,,)

Your formula will now read:

if(g3>75,"A",if(,,))

Make sure that you have the correct number of brackets at the end of the formula. There should be two closing brackets next to each other. The next stage is to enter the new condition, the first parameter to the second if() function. This time we will test that the mark is greater than or equal to 50. Type between the opening bracket and first comma:

g3>=50

There’s no need to test that the mark is over 75, because we have already assigned a grade to those marks and this calculation will not be performed on marks which met the condition in the first part of the function. Your formula now reads:

if(g3>75,"A",if(g3>=50,,))

The final stage is to enter what will be done if this second condition is true or false. If it is true (ie the mark is between 50 and 75) we want to print the grade B; if it is false, the mark will be neither over 75 nor 50 or over, so it must be under 50, and receive the grade C. So the second parameter will be "B" and the third will be "C". Type these in. The finished function should look like this:

if(g3>75,"A",if(g3>=50,"B","C"))

To evaluate the formula, you need to tell Fireworkz that you have finished typing it in. Click the Tick button at the left of the formula line. You should see the result A appear in the cell containing the formula. Note that the quotes are not displayed.

Finally, we need to perform the same calculation for each student in the class. Again, we can use the Fill down button to do this. Select cell h3 and drag down to cell h18. The column will fill with the grades for each student:
Analysing information

We now need to analyse the information on the grades for the class as a whole. We want to count the number of students with grade A, grade B and grade C.

The easiest way to do this is to use one of the database functions. Database functions make a selection from a set of records and perform a calculation on the selection.

We need to find the students with each grade, and simply count how many there are, so the dcounta() function is the one to use. Click on cell b23, and click on the formula line to place the caret there.

Click on the Function selector button, and then click on Categories. Choose Database, then find and double-click on the function dcounta(range,condition).

This function has two parameters; the first gives the range of cells on which the function is to operate, and the second the condition to be applied to them.

In this case the range is the cells containing grades, cells h3 to h18. So type this as the first parameter, but place a dollar sign before the number. After the comma type the condition:

h$3h$18="A"

Note that we need to enter the quote marks around the text string for Fireworkz to recognise it.

Inserting the dollar signs in the cell reference means that the row references will not be changed when the formula is copied using Fill down, but will always refer to the row containing the students and their marks.

The finished formula should look like this:

dcounta(h$3h$18,h$3h$18="A")

Click the Tick button or press Return. The answer 2.00 should appear in cell B23.

Select cells b23 to b25 and click the Fill down button.

In cell b24 we are looking for students with grade B, and in cell b25 looking for students with grade C. Edit the formulas in these cells to contain the correct letter. The formulas will be identical to that in b23 but the final text string (ie the grade) will differ.
Using operators

As a finishing touch, we will display the information about students in each grade category as a percentage of the whole.

There are two parts to this process. First we need to calculate the percentage, then to use a percentage style to format and display the result correctly.

It’s easy to calculate a percentage; simply divide by the total number of elements in the group.

For simple arithmetic operations, we use the arithmetic operators +, -, * and /.

Click on cell c23, and click in the formula line. Click on cell b23. You’ll see b23 appear in the formula line. We need to divide this number by the total number of students, which we can work out using the counta() function. Now type in

/counta(a$3a$18)

and click the Tick button. The result 0.13 will appear in the cell.

Inserting the dollar signs in the cell reference means that the row references will not be changed when the formula is copied using Fill down, but will always refer to the row containing the students and their marks.

Select cells c23 to c25 and click Fill down to calculate the remaining percentages.

With the cells still selected, click the Style button. Choose Percentage from the style list, and click Apply. The numbers will be displayed as a percentage; in this case, this means that they are multiplied by 100, and shown with no decimal places and a percentage sign:

Updating information

The advantage of using a spreadsheet instead of pencil and paper is the ease with which information can be recalculated. After all the marks have been gathered and the analysis completed, news comes in that the external moderators have changed the marks for some English students.

Click on cell b3. Type in 65 over the number. This changes the average score to under 75, and so changes the Grade, the number of students with each grade and the percentage of students with each grade.

Click the Tick button to confirm the new cell contents. All the calculations will be updated straightaway.
