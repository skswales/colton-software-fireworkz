Introduction

About Wordz, Resultz and Fireworkz

Wordz, Resultz and Fireworkz are part of the Z family of applications from Colton Software.

You can use Wordz alone as a word processor to write letters, reports, books, invitations, shopping lists, schedules and other documents.

You can use Resultz alone as a spreadsheet to perform calculations, analyse figures, and produce colourful charts.

Fireworkz combines these two members of the family enabling you to do even more—incorporate live results in your reports, or automatic mail-merge to your friends or customers, for example.

Once you have learned to use one of these programs you will find it easy to use the other members of the family, because they share much of the same interface and have been designed to work well together. This is true even if you move from RISC-OS to Windows or vice versa; the programs have been designed to be as similar as possible on each platform.
About this Getting Started Guide

This Getting Started Guide includes all the information you need to install and start using Colton Fireworkz. You can find more information about the program and its features in the on-line help and in the User Guide and Spreadsheet Guide.

This Getting Started Guide includes:

A brief installation guide

Tutorial files to show you the main features of the program

Hints and tips for transferring files between the RISC OS and Windows versions of Fireworkz.

What you need to use these programs

This Getting Started Guide assumes that you are already reasonably familiar with the way your computer works. If you are new to computing, or to Windows, you are advised to familiarise yourself both with the relevant terminology and the basics of using your computer. You’ll need to know how to:

Start up programs and open documents;

Access a disc, view directories, and copy files;

Display menus and choose entries from them;

Position and resize windows, use the Program Manager, minimise and restore files (Windows)

Scroll through documents using the scroll bars and arrows;

Use the mouse to move the pointer around the screen and to display and position the caret, and how to use the mouse buttons to click, double-click and drag objects.

You should also understand terms such as directory, double click, and dialogue box.

You’ll find all this information in the Windows documentation which came with your computer. The tutorial chapters in the Getting Started Guide also provide a reminder of these skills.

Text in this Getting Started Guide

When working your way through the tutorial chapters, you will see that certain stylistic conventions are used to make it easier for you to see what you should do at each stage, what you should type in and what you should choose from menus.

Menu commands which appear in the text are always given in bold type; for example ‘Choose the Print command.’ So are action buttons in dialogue boxes, such as the OK button.

In some places there is text for you to type in. This is always shown in a typewriter-style font, like this:

Type this text on your computer exactly as you see it.

When the text given represents what you will type—for example, you need to give a file a name before saving it—this is shown by using italics, such as:

Type filename in the dialogue box which appears.

There are keyboard shortcuts which you can use in Fireworkz.

The control key is referred to as Ctrl. If you see a key combination such as Ctrl-X or Alt-space, it means you should press the keys together in the following way:

Press and hold down the first key of the pair.

Press and release the second key.

Release the first key.

You will see many references to the caret. This is the special pointer which shows you where you are in a document, and where any new text you add will be placed.

Some commands can be accessed directly by pressing a function key. Where this is the case the function key is referred to as function key F1, as shown on the key cap itself.
