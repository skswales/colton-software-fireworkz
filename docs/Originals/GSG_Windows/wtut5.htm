Working with Fireworkz

Introduction

This chapter introduces several aspects of Fireworkz. By working through this chapter you will learn to:

Add a date

Use the style editor

Use the ruler to change margins

Add a picture to a document

Set up and perform mailmerge
Editing and formatting text

Load the document saleslet.fwk from the Examples directory. You will see that it contains a letter from a company which is a standard reply to customers who have enquired about prices and products.

The document contains a picture, the name of the company in coloured letters up the side of the page, the company’s address, the text of the letter, and at the bottom of the page, a table.

The picture is in the wrong place. It is in the centre of the page, when it should be aligned with the left edge of the text.

Click in the area of the picture. A red outline will appear. Click inside this outline and drag the picture to the left, till its left edge aligns with the text.
Adding a date

The next step is to add the date to the letter. You can add dates in a wide range of formats, to save you typing it in.

Click in the line above the first line of text (add1) at the left edge of the page.

Click the Right justify button; the caret will move to the right edge of the page.

Choose Date from the Inserts menu (on the Extra menu). The following dialogue box will appear:

Click the last format in the list, which will give the date as a number followed by month and year. Fireworkz always enters today’s date, and shows it in the dialogue box.

Click the button marked Live. Whenever you open this document, the current date will always be used. If this option is switched off, the date entered today will stay fixed in the document.

Click OK. The date will be inserted into the document.
Mailmerge

When this letter is sent to customers, it is personalised to some extent. As enquiries arrive they are logged in a worksheet, also held in Fireworkz. At the end of the day, all the enquiries are printed using the mail shots facility.

Load the document names.fwk from the Tutorial directory. You’ll see a list of names and addresses held in the rows of a worksheet; the second row contains the title of each column.

Selecting the database

The first stage in setting up mailmerge is to choose the database holding the names and addresses to be used. This is held in the Names worksheet. (You can use any Fireworkz, Resultz, or CSV document for this purpose). To tell Fireworkz which document to use for mailmerge:

Make sure that saleslet.fwk is the active document.

Choose Mailshot from the Extra menu, and choose Select... from the Mailshot submenu. A dialogue box will appear:

You need to select the file to use. Click on names.

Next you need to select the row which will be used to show you what is happening when you add fields to your letter. Choose row 1 as this is the one containing the titles of the columns.

Click OK.

Adding fields to the letter

You are now ready to add the fields to your document. To do this:

Select the first line of the recipient’s address (add1) in the saleslet document.

Choose Mailshot from the Extra menu, and Insert field from the Mailshot menu. A dialogue box will appear.

You need to enter the number of the column holding the information you want to appear here. The first address field in the Names document is column 4, so enter 4 in the dialogue box and click OK.

In the sales letter document, you will see that the contents of the cell in column 4 and row 2 have been added to the document. You will see the text Address1.

Go through the next three lines, adding fields 5, 6 and 7. These should insert text reading Address2, Address3, and Postcode, respectively.

In the salutation line of the letter, you need to add the fields for the title and the name.

Select the word title, and insert field 1.

Select the word name, and insert field 3.
Dealing with blank fields

One problem with mail merge is that addresses in the list have blank fields, which could cause blank lines to appear within the address in the finished letter. Fireworkz lets you deal with this.

Choose Select from the Mailshot menu and display row 5. You’ll see that this introduces a blank line into the document.

Choose Select from the Mailshot menu again, and click the button marked Shuffle. The fields which you want to shuffle are those containing the address, so enter 4 as the start row and 7 as the end row in the dialogue. This tells Fireworkz to move any blank lines that appear in the document between certain fields to the end of the space. Click OK.
Editing the table

The table at the bottom of the page requires a small amount of work doing to it. As you will see, two columns contain text, two columns contain numbers, and the last column contains a formula.

Applying a style

The third and fourth columns of the table contain numbers. These are currently unformatted, but should be displayed as currency as they are prices.

Select the cells in these columns which contain prices.

Choose Styles from the Style menu. A list of styles will appear.

Choose pounds pence from the list.

Click Apply. The numbers will be redisplayed with £ signs. Note that they are also now right justified within their cells.

Creating a style

The numbers displayed in the last column show customers the savings they receive if they take advantage of the special offer. However, these numbers are not currently displayed as a percentage.

A new style is needed to display the numbers as percentages.

Choose Styles from the Style menu, and click New from the menu which appears.

Enter the name percentage for the new style, and click on Base letter as the style for it to be based on.

Click OK. The new style will be added to the list, and you will go to the style editor dialogue box to define the style.

Go to the Ruler section, by clicking on the button at the left marked Ruler. Click the Justify button, and click the Right align option. This means that the numbers to which this style is applied will be right aligned.

Go to the number section, by clicking on the button at the left marked Number. Click the number button, then click the arrow at the right. This will display a list of number formats. Click on the following:

0%

Click OK. The style’s definition will be added to the list.

Select the numbers in the last column and click the Style button. Choose percentage from the list and click Apply.
Summary

In this chapter you learned how to use the mail shot facility, and had more practice at using the Style editor when working with tables, including setting up a new style containing a number format.

For more information on mailmerge and the Style Editor, see the relevant chapters in the User Guide.
