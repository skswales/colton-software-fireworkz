SGN
SGN(number)
Returns -1, 0 or 1, depending on whether number is negative,
zero or positive respectively. So, for example:
SGN( -11)
returns the result -1.

SHIFT
SHIFT(graphic, x-offset, y-offset)
Returns a drawfile graphic translated by x-offset OS units
horizontally and y-offset OS units vertically relative to its origin.
For example:
SHIFT(FILE (Draw1) , 100 , 200)
returns the drawfile Draw1 shifted (100,200) from its origin.

SIN
SIN(angle)
Returns the sine of angle.

SQR
SQR(number)
Returns the square root of number.

STR$
STR$(number)
Converts the number number into a string. For example:
STR$(56)
returns the string 56.

SUM
SUM(numeric-expr {, numeric—expr}*)
Returns a real number formed by adding together all the numeric
arguments from the given list. NULL arguments are ignored. For
example:
SUM(13,2, 61, 103)
returns the value 179.

TAN
TAN(angle)
Returns the tangent of angl e.

TIME
TIME(hour, minute, second)
Returns a date/time from hour, minute and second values. For
example:
TIME(2,3,4)
returns 02:03:04.
