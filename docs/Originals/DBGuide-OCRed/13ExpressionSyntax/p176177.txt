CHR$
CHR$(number)
Returns the text character whose ASCII number is expressed by
number. For example:
CHR$(65)
retums A.

COS
COS(angle)
Returns the cosine of angle.

COUNT
COUNT(expr {, expr}*)
Retums an integer giving the number of non-NULL items in a list
of expressions. If all items are NULL the result is zero. For
example:
COUNT(13,2, 61, 103)
returns the value 4.

DATA
DATE (year, month, day)
Retums a date formed from year, month and day values. For
example:
DATE(95,10,05)
returns the date 5th October 1995.

DATENEW
DATENEW(date, time)
Retums a date/time from a date and time. For example:
DATENEW( 'Date1' , 'Time1')
returns a date/time taking the date components from the value in
the field Date1 and the time components from the value in the
field Time1.

DATEONLY
DATEONLY(date)
Returns the date only (i.e. the year, month and day) from date
which may include time components (i.e. hours, minutes and
seconds). For example:
DATEONLY( ' Date1' )
returns a date from the date/time value in the field Date1.

DATEVALUE
DATEVALUE(String)
Retums a date from the string expression. For example:
DATEVALUE("13th Feb 1994 2:42 am")
retums the date 13.02.94 02:42:00.

DAYNAME
DAYNAME(integer)
Returns a text string giving the name of the day corresponding to
the integer, from Monday (0) to Sunday (6). For example:
DAYNAME (2 )
returns "Wednesday".
