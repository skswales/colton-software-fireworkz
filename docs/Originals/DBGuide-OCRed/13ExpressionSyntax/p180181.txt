INT
INT(number)
Returns the integer value of number. So, for example:
INT(-5.5 )
returns the value -5.

INTERVAL
INTERVAL(years,months,days,hours,
minutes,seconds)
Retums an interval from the values for years, months, days, hours,
minutes and seconds. For example:
INTERVAL(1,2,3,4,5,5)
returns the interval year 2 months 3 days 4
hours 5 minutes 6 seconds.

INTERVALVALUE
INTERVALVALUE(s tring)
Retums a time interval from the string expression. For example:
INTERVALVALUE("1 hour 2 minutes 5
seconds")
returns the interval 1 hour 2 minutes 5 seconds

ITEMNO
ITEMNO(string1, string2 {, string3})
Retums the number of the first item from string1 which matches
string2. String3 is an optional separator, newline is assumed to be
the separator if this is not supplied. ITEMNO retums NULL if
string2 is NULL or not found in string1. For example:
ITEMNO("Ann Ben Carl Dean","Ben"," ")
retums the number 2.

LEFT$
LEFT$(text, number)
Returns the leftmost number characters of text. For example:
LEFT$("abcdefgh",3)
returns the string abc.

LEN
LEN(text)
Retums the number of characters in text. For example:
LEN("Recordz")
retums the value 7.

LN
LN(number)
Returns the natural logarithm (log to the base e) of number.
