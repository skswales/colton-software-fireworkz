ODF.IMARGUMENT

Returns the argument[1] of a complex number as a real number in the range (-π,π] (i.e. exclusive, inclusive).

This is the counter-clockwise angle between the complex number and the positive real axis in radians as a real number.

Syntax:

ODF.IMARGUMENT( complex_number_odf )

Revisions:

This function was added in Fireworkz 2.00.
