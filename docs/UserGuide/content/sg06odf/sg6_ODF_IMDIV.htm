ODF.IMDIV

Divides[1] one complex number by another.

Syntax:

ODF.IMDIV( complex_dividend_odf, complex_divisor_odf )

Returns complex_dividend_odf  ∕  complex_divisor_odf.

Revisions:

This function was added in Fireworkz 2.00.
