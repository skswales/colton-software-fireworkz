ODF.IMEXP

Returns the constant e raised to the power[1] of a complex number.

Syntax:

ODF.IMEXP( complex_number_odf )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Complex Exponentiation." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/ComplexExponentiation.html
