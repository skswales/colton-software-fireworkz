ODF.IMPRODUCT

Multiplies[1] one complex number by another.

Syntax:

ODF.IMPRODUCT( complex_multiplicand_odf_1, complex_multiplicand_odf_2 )

Returns complex_multiplicand_odf_1 × complex_multiplicand_odf_2.

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Complex Multiplication." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/ComplexMultiplication.html
