ODF.MOD

Returns the (OpenDocument / Microsoft Excel-compatible) remainder of a division.

Syntax:

ODF.MOD( dividend:Number, divisor:Number )

ODF.MOD gives the remainder (modulo) of the number dividend divided by the number divisor.

The remainder given by ODF.MOD has the same sign as the divisor. Note that this behaviour is NOT compatible with Fireworkz' MOD for negative numbers.

Revisions:

This function was added in Fireworkz 2.20.
