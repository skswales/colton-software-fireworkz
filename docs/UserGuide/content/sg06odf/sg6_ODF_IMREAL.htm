ODF.IMREAL

Returns the real part[1] of a complex number as a real number.

Syntax:

ODF.IMREAL( complex_number )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Real Part." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/RealPart.html
