ODF.IMABS

Returns the modulus[1] |z| of a complex number as a real number.

Syntax:

ODF.IMABS( complex_number_odf )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Complex Modulus." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/ComplexModulus.html
