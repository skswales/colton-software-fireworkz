ODF.TDIST

OpenDocument / Microsoft Excel compatible function.

Syntax:

ODF.TDIST( x:Number, degrees of freedom:Integer, tails:Integer )

Returns the probability for the Student's t-distribution with given x, degrees of freedom and one or two tails.

See also T.DIST(), T.DIST.2T(), and T.DIST.RT().

Revisions:

This function was added in Fireworkz 2.00.
