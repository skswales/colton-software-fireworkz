ODF.IMPOWER

Syntax:

ODF.IMPOWER( complex_number_odf_1, complex_number_odf_2 )

Returns complex_number_odf_1 raised to the power[1][2] of complex_number_odf_2.

Revisions:

This function was added in Fireworkz 2.00.
