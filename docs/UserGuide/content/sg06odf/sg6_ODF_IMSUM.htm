ODF.IMSUM

Adds[1] one complex number to another.

Syntax:

ODF.IMSUM( complex_addend_odf_1, complex_addend_odf_2 )

Returns complex_addend_odf_1 + complex_addend_odf_2.

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Complex Addition." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/ComplexAddition.html
