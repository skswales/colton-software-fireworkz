Choices dialogue box

There are some settings in Fireworkz which you will want set in every document you produce. They relate to the way you use Fireworkz rather than to individual documents you work on.

These choices are made using the Choices dialogue box, which is accessed from the icon bar menu (RISC OS) or from the File menu (Windows).

It includes options for the following:

Spelling

Auto save

Kerning

Dithering

Pictures

Status line

Default ruler

Recalculation

Charts

Saving your choices

To save the choices you have made in the Choices dialogue box, click on the Save button in the dialogue box. Your choices will be saved as a set of files in the <Choices$Write>.Fireworkz.Choices (RISC OS) or %APPDATA%\Colton Software\Fireworkz\Choices (Windows) directories.
