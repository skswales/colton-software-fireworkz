Spelling

Some spelling checker settings are set via the Choices dialogue box.

Select the Auto check option to turn check as you type spelling checking on.

Select the Load Master dictionary option to load the master dictionary automatically when starting up. This is of most use if you are using floppy discs as you can force Fireworkz to load the dictionary when it starts up.

Select the Load user dictionary option to load the user dictionary.

Select the Write to user dictionary option to allow new words to be added to the user dictionary.

For information on using the spelling checker see the Editing text section.
