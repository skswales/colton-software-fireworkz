Editing header and footer text

You can edit header and footer text on any page on which it appears.

To edit the text:

  1. Place the caret at the place where you want to start editing.
  2. Replace the text or add new text in the usual way.
  3. This process will change the header or footer on all the pages where it appears.
  4. If you want to change the header or footer on some but not all pages, create a new header or footer with the different text and use the Pages dialogue box to apply it to the pages you want.
