Number cells

Cells which contain the data for your calculations must be marked as number cells.

With number cells:

  • You must type in your data or formula on the formula line, and click the Tick button (or press Return) to enter the data or the formula into the cell in the worksheet.
  • Data entered in number cells can be used easily as the basis of calculations.
  • Numerical information you enter on the formula line, or the result of any formula you enter, is formatted according to the Number or Date format defined in the cells style, before being displayed on the worksheet.
  • If you type text on the formula line and Fireworkz is unable to understand it as numerical data or a formula, it will make the cell a text cell, and apply the Text style to it to make the text it contains left justified.
  • You can enter references to other cells as part of your formula. References to text cells will produce a text string.
  • You can't enter information using the Inserts menu.
