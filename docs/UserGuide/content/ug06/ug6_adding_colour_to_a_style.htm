Adding colour to a style

To add colour to a style:

  1. Display the Colours section of the Style editor.
  2. Click on the button for background, border or grid as required.
  3. Choose the colour you want, as described above for text colour.
  4. Turn on grid and borders you must also pick a line style, by clicking the Line style button and then clicking on a line button. There are four to choose from, including thin, standard and thick solid lines and dotted lines. The fifth, blank, button turns off any lines which are already there.
