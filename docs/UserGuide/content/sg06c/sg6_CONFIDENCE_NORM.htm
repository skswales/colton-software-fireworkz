CONFIDENCE.NORM

Confidence interval for a normal distribution.

Syntax:

CONFIDENCE.NORM( alpha:Number, standard deviation:Number, sample size:Positive integer )

Returns the confidence interval for a population mean given significance level alpha in (0,1) and sample size for the normal distribution with given standard deviation (> 0).

Revisions:

This function was added in Fireworkz 2.00.
