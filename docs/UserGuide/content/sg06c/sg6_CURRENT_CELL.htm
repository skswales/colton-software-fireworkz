CURRENT_CELL

Returns the cell reference of the current cell.

Syntax:

CURRENT_CELL

This is a parameterless function (like PI) which returns the cell reference of the current cell. Any cell containing the CURRENT_CELL function is recalculated every time the current cell changes, that is whenever you move around the spreadsheet.

For example, if you put the simple expression CURRENT_CELL into a cell, that cell will display the contents of the current cell as you move around the sheet.
