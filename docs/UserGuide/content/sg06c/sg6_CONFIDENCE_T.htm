CONFIDENCE.T

Confidence interval for a Student's t-distribution.

Syntax:

CONFIDENCE.T( alpha:Number, standard deviation:Number, sample size:Positive integer )

Returns the confidence interval for a population mean given significance level alpha in (0,1) and sample size for the Student's t-distribution with given standard deviation (> 0).

Revisions:

This function was added in Fireworkz 2.00.
