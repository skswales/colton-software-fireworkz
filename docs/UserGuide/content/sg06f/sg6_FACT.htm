FACT

Calculates the factorial[1] of a number.

Syntax:

FACT( number )

Returns the factorial of a number (commonly shortened to n!).

number must be an integer greater than or equal to zero.

Formula:

n! = i ∏1n i

i.e.

n! = n × (n − 1) … × 2 × 1

Revisions:

This function was added in 1.30.
