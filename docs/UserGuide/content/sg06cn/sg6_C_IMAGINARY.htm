C_IMAGINARY

Returns the imaginary part[1] of a complex number as a real number.

Syntax:

C_IMAGINARY( complex_number )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Imaginary Part." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/ImaginaryPart.html
