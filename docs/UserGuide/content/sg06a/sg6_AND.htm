AND

Calculates the Boolean (logical) AND of a number of Boolean values.

Syntax:

AND( Boolean_values:List )

Returns the Boolean (logical) AND of all the Boolean values in the list.

If all parameters passed to the AND function are TRUE, the Boolean value TRUE is returned; otherwise FALSE is returned.

Note:

Unlike the corresponding AND Boolean operator, &, all parameters of the AND function are always evaluated.

Example:

AND(A1, A2)

where the cell values are TRUE and FALSE will return the Boolean value FALSE.

Revisions:

This function was added in Fireworkz 2.00.
