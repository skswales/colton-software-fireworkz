QUARTILE.EXC

Returns a quartile value from a data set.

Syntax:

QUARTILE.EXC( data:Range or Array, q:Number )

Returns the qth quartile from the values in the data range of cells (or array) based on a percentile range of (0,1) (i.e. exclusive).

q must be of one the following values:

1  25% Q1
2  50% Q2 (MEDIAN)
3  75% Q3

Revisions:

This function was added in Fireworkz 2.00.
