GEOMEAN

Calculates the geometric mean[1] of the values in a data set.

Syntax:

GEOMEAN( list )

Returns the geometric mean of the values in the list.

list may be a range of cells, an array, or a comma-separated list of values (which may contain cell references and also ranges of cells as one or more of its elements).

All values in the list must be positive, non-zero numbers.

Formula:

GEOMEAN = ( i ∏1n xi ) 1/n

where n is the number of values and xi is the i‑th value.

Example:

GEOMEAN(A1:A15)

returns the geometric mean of the contents of cells in the range A1:A15.
