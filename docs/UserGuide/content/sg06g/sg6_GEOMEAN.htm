GEOMEAN

Calculates the geometric mean of the values in a data set.

Syntax:

GEOMEAN(list)

Returns the geometric mean of the values in the list.

list may be an array, a range of cells, or a comma-separated list of values.

All values in the list must be positive, non-zero numbers.

Formula:

GEOMEAN = ( ∏i=n xi ) 1/n

where n is the number of values and xi is the i-th value.

Example:

GEOMEAN(A1:A15)

returns the geometric mean of the contents of cells in the range A1:A15.

Revisions:

This function was added in Fireworkz 2.00.
