GAMMALN

Syntax:

GAMMALN( x:Number )

This special function yields the natural logarithm of the absolute value of Γ(x)[1].

Revisions:

This function was added in 1.30.

  1. Weisstein, Eric W. "Log Gamma Function." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/LogGammaFunction.html
