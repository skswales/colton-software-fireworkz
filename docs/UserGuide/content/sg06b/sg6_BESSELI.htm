BESSELI

Returns the modified Bessel function of the first kind[1] of a number. In(x).

Syntax:

BESSELI( x:Number, n:Number )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Modified Bessel Function of the First Kind." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/ModifiedBesselFunctionoftheFirstKind.html
