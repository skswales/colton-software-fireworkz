BESSELJ

Returns the Bessel function of the first kind[1] of a number. Jn(x).

Syntax:

BESSELJ( x:Number, n:Number )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Bessel Function of the First Kind." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/BesselFunctionoftheFirstKind.html
