BIN2OCT

Converts a binary number to its corresponding octal representation.

Syntax:

BIN2OCT( number {, places:Number} )

number may be a string, or a number, of up to ten binary digits.

Two's complement representation is used: the highest (tenth) bit is the sign bit of the binary number and the highest (thirtieth) bit is the sign bit of the octal result.

If specified, places must be in the range 1..10. If places is omitted, the minimum appropriate number of digits is returned.

Examples:

BIN2OCT("11111111", 4)

returns the text string "0377".

BIN2OCT("1111111111")

returns the text string "7777777777".

Revisions:

This function was added in Fireworkz 2.24.
