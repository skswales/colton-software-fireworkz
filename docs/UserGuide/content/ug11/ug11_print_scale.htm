Print scale

This option lets you set the size of the pages you print, so that you can print reduced or enlarged pages. To set it:

  1. Choose Scale from the Page menu.
  2. Select the options required from the Scale dialogue box.
    • Fit one page will fit your page to the paper, reducing or enlarging as necessary.
    • Fit one page x will do the same, but with the paper oriented in landscape mode.
    • 100% will print the document at its true size. You can also enter a custom percentage.
  3. Click OK. Until you change this setting, this is how the current document will be printed.
