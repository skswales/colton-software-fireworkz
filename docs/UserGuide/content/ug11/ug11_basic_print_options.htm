Basic print options

When you choose Print from a menu or the toolbar, the basic Print dialogue box will appear. Its title bar will show you the type of printer driver you have loaded (e.g. PostScript).

  1. Enter the number of copies of your document that you want to print.
  2. If you only want to print some of the pages in your document, select the Range radio button. You will then be able to type in the number of the page you want to start printing from, and the number of the last page in the range you want to print.
  3. Click OK to print.
