14.7 Mathematical functions

A range of mathematical functions are provided by Fireworkz.

  • ABS( number )
  • BASE( number,  radix {, minimum_length} )
  • CEILING( number {, multiple} )
  • DECIMAL( number,  radix )
  • EVEN( number )
  • EXP( number )
  • FACT( number )
  • FACTDOUBLE( number )
  • FLOOR( number {, multiple} )
  • GCD( list )
  • INT( number )
  • LCM( list )
  • LN( number )
  • LOG( number { base} )
  • MOD( number, divisor )
  • MROUND( number, multiple )
  • MULTINOMIAL( list )
  • ODD( number )
  • POWER( number, power )
  • PRODUCT( list )
  • QUOTIENT( numerator, denominator )
  • RAND( )
  • RANDBETWEEN( lower, upper )
  • ROUND( number, digits )
  • ROUNDDOWN( number, digits )
  • ROUNDUP( number, digits )
  • SERIESSUM( x, n, m, coefficients )
  • SGN( number )
  • SQR( number )
  • SUM( list )
  • SUMPRODUCT( array_1 … array_N )
  • SUMSQ( list )
  • SUM_X2MY2( x‑data, y‑data )
  • SUM_X2PY2( x‑data, y‑data )
  • SUM_XMY2( x‑data, y‑data )
  • TRUNC( number {, digits} )
