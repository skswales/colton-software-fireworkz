14.13 Compatibility functions

These functions are most useful for OpenDocument / Microsoft Excel compatibility (e.g. differing parameter order than Fireworkz).

Engineering compatibility functions

These compatibility complex number functions provide arithmetic, mathematical and trigonometrical functions for use with complex number strings, i.e. text strings representing complex numbers that have both real and imaginary components.

In Fireworkz, complex numbers passed to compatibility complex number functions should be expressed as OpenDocument / Microsoft Excel-style complex number strings.

  • ODF.COMPLEX( real part, imaginary part {, suffix} )
  • ODF.IMABS( complex_number_odf )
  • ODF.IMAGINARY( complex_number_odf )
  • ODF.IMARGUMENT( complex_number_odf )
  • ODF.IMCONJUGATE( complex_number_odf )
  • ODF.IMCOS( complex_number_odf )
  • ODF.IMDIV( complex_dividend_odf, complex_divisor_odf )
  • ODF.IMEXP( complex_number_odf )
  • ODF.IMLN( complex_number_odf )
  • ODF.IMLOG10( complex_number_odf )
  • ODF.IMLOG2( complex_number_odf )
  • ODF.IMPOWER( complex_number_odf_1, complex_number_odf_2 )
  • ODF.IMPRODUCT( complex_multiplicand_odf_1, complex_multiplicand_odf_2 )
  • ODF.IMREAL( complex_number_odf )
  • ODF.IMSIN( complex_number_odf )
  • ODF.IMSQRT( complex_number_odf )
  • ODF.IMSUB( complex_minuend_odf, complex_subtrahend_odf )
  • ODF.IMSUM( complex_addend_odf_1, complex_addend_odf_2 )

Financial compatibility functions

  • ODF.FV( rate, nper, pmt )
  • ODF.IRR( cashflows {, guess} )
  • ODF.PMT( rate, nper, pv )

Mathematical compatibility function

  • ODF.INT( number )
  • ODF.LOG10( number )
  • ODF.MOD( number, divisor )

Miscellaneous compatibility functions

  • ODF.INDEX( data, row, column )
  • ODF.TYPE( parameter )

Statistical compatibility functions

  • ODF.BETADIST( x, alpha, beta {, lower_limit {, upper_limit {, cumulative} } } )
  • ODF.TDIST( x, degrees of freedom, tails )
