14.8 Matrix functions

Matrix functions work internally on arrays and return an array (or a number, in the case of M_DETERM).

  • M_DETERM( matrix )
  • M_INVERSE( matrix )
  • M_MULT( matrix_1, matrix_2 )
  • M_UNIT( number )
  • TRANSPOSE( matrix )
