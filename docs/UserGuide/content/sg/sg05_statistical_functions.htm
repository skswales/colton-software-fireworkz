14.10 Statistical functions

Many statistical functions can operate on a range of cells or an array.

In those cases where a list is specified as the parameter the function can also take a comma-separated list of values. A list may contain cell references and also ranges of cells as one (or more) of their elements.

  • AVEDEV( list )
  • AVERAGEA( list )
  • AVG( list )
  • BETA( a, b )
  • BETA.DIST( x, alpha, beta {, cumulative {, lower_limit {, upper_limit} } } )
  • BETA.INV( probability, alpha, beta {, lower_limit {, upper_limit} } } )
  • BIN( data, bins )
