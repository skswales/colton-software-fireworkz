14.11 String functions

String functions operate on text strings. If you deliberately enter a string into a number cell, remember to enclose it in double quotes. Unlike other most functions, string functions can also operate on text cells.

  • CHAR( number )
  • CLEAN( text )
  • CODE( text )
  • DOLLAR( number {, digits} )
  • EXACT( a, b )
  • FIND( search, source {, start} )
  • FIXED( number {, digits {, no_commas} } )
  • FORMULA_TEXT( cell_reference )
  • JOIN( a, b … )
  • LEFT( text {, length} )
  • LENGTH( text )
  • LOWER( text )
  • MID( text, start, length )
  • PROPER( text )
  • REPLACE( text, start, length, new )
  • REPT( text, number )
  • REVERSE( text )
  • RIGHT( text {, length} )
  • STRING( number, digits )
  • SUBSTITUTE( text, old, new {, instance} )
  • T( value )
  • TEXT( value or Date, format )
  • TRIM( text )
  • UPPER( text )
  • VALUE( text )
