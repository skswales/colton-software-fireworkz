14.2 Database functions

Database functions provide a set of powerful tools for data analysis.

They perform a mathematical or statistical function on the values of only those cells in a range which meet a corresponding specified condition. Unlike those corresponding functions, database functions cannot operate on an array as the condition must refer to another range within, or parallel to, cells in the range.

  • DAVG( range, condition )
  • DCOUNT( range, condition )
  • DCOUNTA( range, condition )
  • DMAX( range, condition )
  • DMIN( range, condition )
  • DPRODUCT( range, condition )
  • DSTD( range, condition )
  • DSTDP( range, condition)
  • DSUM( range, condition)
  • DVAR( range, condition )
  • DVARP( range, condition )
