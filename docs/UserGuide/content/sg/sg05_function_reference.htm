14 Spreadsheet function reference

Fireworkz has a large number of built-in functions for use in spreadsheets:

  • Boolean (logical) functions
  • Database functions
  • Date and time functions
  • Engineering functions
  • Financial functions
  • Lookup functions
  • Mathematical functions
  • Matrix functions
  • Miscellaneous functions
  • Statistical functions
  • String functions
  • Trigonometrical functions
  • Compatibility functions

These functions are grouped by category, and shown together with their required parameters and any optional parameters (shown in curly brackets).

Many function parameters listed are of type Number, except where inferred by the parameter name or explicitly specified; but please always check with the parameters specified in the definition of any specific function of interest.
