ERFC

Returns the complementary error function[1] integrated between a lower limit and +infinity.

Syntax:

ERFC( lower_limit:Number )

Revisions:

This function was added in Fireworkz 2.00.

  1. Weisstein, Eric W. "Erfc." From MathWorld--A Wolfram Web Resource. https://mathworld.wolfram.com/Erfc.html
