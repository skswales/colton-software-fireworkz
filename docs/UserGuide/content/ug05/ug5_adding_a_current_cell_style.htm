Adding a current cell style

A current cell style is not automatically added to a document based on a letter template. The easiest way to add one is to:

  1. Open a document containing a current cell style.
  2. Choose Save template from the File menu.
  3. Select Style and click on the Current cell style in the list of styles.
  4. Drag the template icon on to the document to which you wish to add it.
  5. Your document will now have a current cell style.
