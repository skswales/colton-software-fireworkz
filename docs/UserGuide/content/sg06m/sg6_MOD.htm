MOD

Returns the remainder of a division.

Syntax:

MOD( dividend:Number, divisor:Number )

MOD gives the remainder (modulo) of the number dividend divided by the number divisor.

The remainder given by MOD has the same sign as the dividend. Note that this behaviour is NOT compatible with OpenDocument / Microsoft Excel MOD for negative numbers. If you require compatibility with OpenDocument / Microsoft Excel, use ODF.MOD().

Example:

MOD(7, 3)

returns the number 1.
