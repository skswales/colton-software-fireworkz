Database functions

Database functions provide a set of powerful tools for data analysis.

They perform a mathematical or statistical function on the values of only those cells in a range which meet a corresponding specified condition.

  • DAVG
  • DCOUNT
  • DCOUNTA
  • DMAX
  • DMIN
  • DPRODUCT
  • DSTD
  • DSTDP
  • DSUM
  • DVAR
  • DVARP
