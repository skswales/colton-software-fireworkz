DEC2BIN

Converts a decimal value to its corresponding binary representation.

Syntax:

DEC2BIN(number {, places:Number})

number may be a positive number or a string of decimal digits.

Two's complement representation is used: the highest (tenth) bit is the sign bit of the binary result.

If specified, places must be in the range 1..10. If places is omitted, the minimum appropriate number of digits is returned.

Examples:

DEC2BIN(16, 8)

returns the text string "00010000".

DEC2BIN(-1)

returns the text string "1111111111".

Revisions:

This function was added in 2.24.
