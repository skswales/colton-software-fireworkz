DVAR

Calculates the sample variance of the values in a data set created from a condition.

Syntax:

DVAR(range, condition)

Returns the sample variance of all the values in the range for which the corresponding condition is satisfied.

Formula:

DVAR = (n⋅∑i=n (xi2) − (∑i=n xi)2) ÷ (n⋅(n − 1))

where n is the number of selected values and xi is the i-th value from the selection.

Example:

DVAR(A1:A15, B1:B15>0)

returns the variance of the contents of cells in the range A1:A15 whose equivalent in the range B1:B15 is greater than zero.

See DAVG for more examples of how to use the condition to control which cells are included.
