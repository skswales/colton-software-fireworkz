DEREF

Syntax:

DEREF( cell_reference )

Returns the value of the cell referred to by cell_reference.

Example:

DEREF(A1)

returns the result of any calculation or formula contained in A1.

This is most usefully combined with the TYPE function. TYPE(A1) will return the text string "reference" because A1 is a cell reference, whereas TYPE(DEREF(A1)) will return the type of data contained in cell A1. This is a necessary step as when Fireworkz recalculates, it delays the evaluation of cell references for as long as possible, in order to avoid unnecessary calculations.
