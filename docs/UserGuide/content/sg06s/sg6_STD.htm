STD

Calculates the sample standard deviation of the values in a data set.

This is the square root of the sample variance.

Syntax:

STD(list)

Returns the sample standard deviation of the values in the list.

list may be an array, a range of cells, or a comma-separated list of values.

Formula:

STD = √ ( (n⋅∑i=n (xi2) − (∑i=n xi)2) ÷ (n⋅(n − 1)) )

where n is the number of values and xi is the i-th value.

Example:

STD(A1:A15)

returns the standard deviation of the contents of cells in the range A1:A15.
