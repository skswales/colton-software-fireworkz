STANDARDIZE

Standarises a value to a normal distribution.

Syntax:

STANDARDIZE( x:Number, mu:Number, sigma:Number )

Standarises the value x to the normal distribution given by mean mu and standard deviation sigma.

Formula:

Z = (x − mu)  ∕  sigma

Example:

SET_VALUE(B1:B15, STANDARDIZE(A1:A15, C1, D1))

returns, in cells B1:B15, the corresponding standarised values of each the contents of cells in the range A1:A15 from the given distribution's mean (C1) and standard deviation (D1) values.

Revisions:

This function was added in Fireworkz 2.00.
