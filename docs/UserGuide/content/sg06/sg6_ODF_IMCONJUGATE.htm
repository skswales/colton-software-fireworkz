ODF.IMCONJUGATE

Returns the complex conjugate[1] of a complex number.

Syntax:

ODF.IMCONJUGATE(complex_number_odf)

Revisions:

This function was added in 2.00.

  1. Weisstein, Eric W. "Complex Conjugate." From MathWorld--A Wolfram Web Resource. http://mathworld.wolfram.com/ComplexConjugate.html
