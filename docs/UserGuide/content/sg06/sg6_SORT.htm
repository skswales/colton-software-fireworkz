SORT

Sorts rows of data in ascending order on the values contained in a given column.

Syntax:

SORT(array, column_index:number)

Sorts rows of data in the array (or range of cells) in ascending order based on the values in the column specified by column_index.

Example:

SORT(B1:E100, 2)

sorts data from the range B1:E100 by row based on the contents of column C. Note that it does NOT affect the content of the source range.

Use FLIP on the result of SORT to sort in descending order.

Use TRANSPOSE prior to SORT to sort by column based on values in rows.
