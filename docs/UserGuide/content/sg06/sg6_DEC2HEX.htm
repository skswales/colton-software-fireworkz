DEC2HEX

Converts a decimal value to its corresponding hexadecimal representation.

Syntax:

DEC2HEX(number {, places:Number})

number may be a positive number or a string of decimal digits.

Two's complement representation is used: the highest (fortieth) bit is the sign bit of the hexadecimal result.

If specified, places must be in the range 1..10. If places is omitted, the minimum appropriate number of digits is returned.

Examples:

DEC2HEX(4096, 8)

returns the text string "00001000".

DEC2HEX(-1)

returns the text string "FFFFFFFFFF".

Revisions:

This function was added in 2.24.
