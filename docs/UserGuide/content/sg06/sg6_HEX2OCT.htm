HEX2OCT

Converts a hexadecimal number to its corresponding octal representation.

Syntax:

HEX2OCT(number {, places:Number})

number may be a string of up to ten hexadecimal digits or a number of up to ten digits.

Two's complement representation is used: the highest (fortieth) bit is the sign bit of the hexadecimal number and the highest (thirtieth) bit is the sign bit of the octal result.

If specified, places must be in the range 1..10. If places is omitted, the minimum appropriate number of digits is returned.

Examples:

HEXOCT("688")

returns the text string "3210".

HEX2OCT("00FF", 8)
;

returns the text string "00000377".

Revisions:

This function was added in 2.24.
