C_SUB

Subtracts[1] one complex number from another.

Syntax:

C_SUB(complex_minuend, complex_subtrahend)

Returns complex_minuend − complex_subtrahend.

  1. Weisstein, Eric W. "Complex Subtraction." From MathWorld--A Wolfram Web Resource. http://mathworld.wolfram.com/ComplexSubtraction.html
