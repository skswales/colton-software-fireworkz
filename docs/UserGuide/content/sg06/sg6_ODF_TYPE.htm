ODF.TYPE

Syntax:

ODF.TYPE(parameter)

Returns an OpenDocument / Microsoft Excel-compatible number corresponding to the type of parameter.

Possible return values are:

  • 1   Number (includes dates and blank cells)
  • 2   Text
  • 4   Logical value
  • 16  Error
  • 64  Array

Revisions:

This function was added in 2.00.
