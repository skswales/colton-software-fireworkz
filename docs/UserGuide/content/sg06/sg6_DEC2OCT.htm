DEC2OCT

Converts a decimal value to its corresponding octal representation.

Syntax:

DEC2OCT(number {, places:Number})

number may be a positive number or a string of decimal digits.

Two's complement representation is used: the highest (thirtieth) bit is the sign bit of the octal result.

If specified, places must be in the range 1..10. If places is omitted, the minimum appropriate number of digits is returned.

Examples:

DEC2OCT(688, 6)

returns the text string "003210".

DEC2OCT(-1)

returns the text string "7777777777".

Revisions:

This function was added in 2.24.
