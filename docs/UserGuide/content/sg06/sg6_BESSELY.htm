BESSELY

Returns the Bessel function of the second kind[1] of a number. Yn(x).

Syntax:

BESSELY(x, n)

Revisions:

This function was added in 2.00.

  1. Weisstein, Eric W. "Bessel Function of the Second Kind." From MathWorld--A Wolfram Web Resource. http://mathworld.wolfram.com/BesselFunctionoftheSecondKind.html
