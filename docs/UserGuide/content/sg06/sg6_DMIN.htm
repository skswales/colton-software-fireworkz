DMIN

Syntax:

DMIN(range, condition)

Returns the minimum value in the range for which the corresponding condition is satisfied.

Example:

DMIN(A1:C5, A1:C5>5 & A1:C5<70)

will return the smallest value from the range of cells A1:C5 which is greater than 5 and smaller than 70.

See DAVG for more examples of how to use the condition to control which cells are included.
