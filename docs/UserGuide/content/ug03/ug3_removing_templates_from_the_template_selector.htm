Removing templates from the template selector

RISC OS: Use the Filer to access the directory
Choices:Fireworkz.Template (for user-defined templates) or
!Fireworkz.AppData.UK.Template (for supplied templates)
and delete the template files you no longer want, or rename them as desired.

Windows: Use the File Explorer to access the directory
%APPDATA%\Colton Software\Fireworkz\Template
and delete the template files you no longer want, or rename them as desired.
